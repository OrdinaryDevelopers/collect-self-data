## 收集自己的数据

### IDEA

只要用手机 OR 电脑的, 都在各种软件上产生了大量的数据, 如果可以将这些数据全部拿到的话那我们可以做很多事情, 分析自己的购物、支出、乘车旅行、居住了哪些酒店、叫了什么外卖、看了什么电影、睡眠、运动、记单词、阅读...将自己产生的所有的数据汇总起来, 可以加以分析, 从而使自己成为一个更好的人。 也可以作为一个历史快照, 记录某个时间点做了什么事等等...
